# 감사 길잡이 (웹·비설치판)

감사원 공개문 기반 **감사 사례 길잡이** — 사례 탐색·분석·내 사례함·메모. 100% 정적 웹앱이라 설치 없이 브라우저에서 바로 실행됩니다.

## 실행 방법

### 1) 온라인 (즐겨찾기 → 클릭)
GitHub Pages 주소를 즐겨찾기에 넣고 클릭하면 실행됩니다. (인터넷 연결 필요)

### 2) 오프라인 / 폐쇄망 (로컬)
- `시작.bat` 더블클릭 → 로컬 서버(`127.0.0.1:8787`)로 실행 (Node.js 필요, 외부 통신 0)
- 또는 `index.html` 더블클릭 (간단하지만 `file://` 보안정책상 일부 저장 기능이 불안정)

## 단축키
- **S** : 현재 선택 사례 내 사례함 담기/빼기 (★⇄☆)
- **W** : 현재 선택 사례 넓게 보기 / 팝업 닫기
- ☆·⤢ 버튼에 마우스를 올리면 단축키 안내가 표시됩니다.

## 참고
- 데이터: 감사원 공개문(공공누리/KOGL 기반).
- 위반법령 "인앱 조문 팝업"은 브라우저 보안정책(CORS)상 지원되지 않으며, 클릭 시 국가법령정보센터로 이동합니다.
