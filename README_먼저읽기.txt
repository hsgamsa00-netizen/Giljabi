감사 길잡이 — 비설치(브라우저)판  ※ 시제품(프로토타입)
====================================================

[실행 방법]
 1) 권장:  "시작.bat" 더블클릭
      → 로컬 서버(127.0.0.1:8787)가 뜨고 기본 브라우저로 자동 접속됩니다.
      → 설치·인터넷 연결이 필요 없습니다(외부 통신 0). 사내 망분리 환경에서도 동작.
      → 종료: 최소화된 "감사 길잡이 서버" 창을 닫으면 됩니다.
      ※ Node.js가 설치돼 있어야 합니다(이 PC에는 이미 설치됨).

 2) 간단 실행: "index.html"을 더블클릭(file://)
      → 대부분 동작하지만, 일부 브라우저에서 file:// 보안정책 때문에
        설정·내 사례함·메모 저장(localStorage)·복사(clipboard)가 불안정할 수 있습니다.
      → 안정적으로 쓰려면 1)번(시작.bat)을 권장합니다.

[새 기능 — 키보드 단축키]
 · S  : 현재 선택한 사례를 "내 사례함"에 담기 / 빼기 (★⇄☆ 토글)
 · W  : 현재 선택한 사례 "넓게 보기"(팝업) / 팝업이 열려 있으면 닫기
 · 사례를 먼저 클릭해 선택한 뒤 눌러야 합니다.
 · 검색창·입력창에 글자를 칠 때는 단축키가 동작하지 않습니다(오타 방지).
 · ☆ 담기 / ⤢ 넓게보기 버튼 위에 마우스를 올리면 단축키 안내가 표시됩니다.

[설치판과 다른 점]
 · "위반 법령 → 인앱 조문 팝업"은 브라우저(비설치)에서는 지원되지 않습니다.
   (법제처 OpenAPI는 브라우저 보안정책(CORS)상 직접 호출 불가)
   대신 법령을 클릭하면 "국가법령정보센터"로 새 창 이동합니다(기존 폴백과 동일).
 · 그 외 사례 탐색·분석·내 사례함·메모·다크모드 등은 설치판과 동일하게 동작합니다.

[데이터]
 · 사례 데이터(cards_data.js)와 모든 자산이 이 폴더 안에 포함되어 있습니다.
 · 폴더째 USB 등으로 복사하면 다른 PC에서도 그대로 실행됩니다.
